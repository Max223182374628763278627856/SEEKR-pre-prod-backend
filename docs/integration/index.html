Guide d'intégration

SEEKR sur votre site
en 5 minutes chrono

Une seule ligne de code. Compatible avec tous les CMS du marché. Choisissez votre plateforme ci-dessous et suivez le guide — votre GPS Sémantique sera actif avant la fin de votre café.

Vue d'ensemble

4 étapes pour tous les CMS

🔑

02
📋

Intégration WordPress

Compatible avec tous les thèmes WordPress — Elementor, Divi, Avada, thèmes enfants.
1
Installez l'extension "Insert Headers and Footers"
Depuis votre admin WordPress, allez dans Extensions → Ajouter, recherchez "Insert Headers and Footers" (WPCode) et installez-la. C'est la méthode recommandée — elle ne modifie pas les fichiers de votre thème.
2
Accédez à l'éditeur de scripts
Dans votre menu WordPress, allez dans Outils → Insert Headers and Footers. Vous verrez deux zones : Header et Footer.
3
Collez le snippet dans la zone Footer
Copiez le snippet SEEKR depuis votre dashboard et collez-le dans la zone Footer. Cliquez sur Enregistrer.
<!-- SEEKR GPS Sémantique -->
<div id="seekr-search"></div>
<script async
  src="https://api.seekr-search.fr/widget/seekr-widget.js"
  data-key="sk_seekr_live_VOTRE_CLE_API"
  data-backend="https://api.seekr-search.fr"
  data-theme="dark-gold"
  data-placeholder="Rechercher sur votre site…"
  data-max-results="8"
  data-gps="true"
></script>
4
Vérifiez l'affichage et lancez le crawl
Rechargez votre site WordPress — le widget SEEKR doit apparaître. Retournez dans votre dashboard SEEKR et cliquez Lancer le crawl avec l'URL de votre site.
💡 Alternative sans extension : Allez dans Apparence → Éditeur de thème → footer.php et collez le snippet juste avant </body>. Attention : cette méthode peut être écrasée lors d'une mise à jour du thème.
🔌 Compatible avec : Elementor, Divi, Avada, OceanWP, Astra, GeneratePress, WooCommerce, tous les thèmes enfants.
⚡ Temps d'intégration estimé : 3 minutes avec l'extension, 5 minutes en éditant footer.php.
🛒

Intégration PrestaShop

Compatible PrestaShop 1.7 et 8.x — idéal pour les e-commerces avec de nombreuses références.
1
Accédez à l'éditeur de thème PrestaShop
Dans votre back-office PrestaShop, allez dans Design → Éditeur de thème. Sélectionnez votre thème actif dans la liste.
2
Ouvrez le fichier footer.tpl
Dans l'arborescence de fichiers, naviguez jusqu'à templates/_partials/footer.tpl (ou footer.tpl selon votre thème). Cliquez pour l'ouvrir dans l'éditeur.
3
Collez le snippet avant </body>
Trouvez la balise </body> dans le fichier et collez le snippet SEEKR juste avant. Sauvegardez le fichier.
<!-- SEEKR GPS Sémantique -->
<div id="seekr-search"></div>
<script async
  src="https://api.seekr-search.fr/widget/seekr-widget.js"
  data-key="sk_seekr_live_VOTRE_CLE_API"
  data-backend="https://api.seekr-search.fr"
  data-theme="dark-gold"
  data-gps="true"
></script>
4
Videz le cache PrestaShop
Allez dans Paramètres avancés → Performance → Vider le cache. Rechargez votre boutique — le widget SEEKR doit apparaître. Lancez ensuite le crawl depuis votre dashboard SEEKR.
💡 Pourquoi SEEKR est parfait pour PrestaShop : La recherche native de PrestaShop cherche uniquement par mot exact. SEEKR comprend "serrure appartement sécurisé" même si vous avez indexé "serrure 3 points A2P". Idéal pour les catalogues de 1 000+ références.
🔌 Compatible avec : PrestaShop 1.7.x, 8.x, tous les thèmes (Classic, Warehouse, Transformer, etc.), WooCommerce modules.
⚡ Temps d'intégration estimé : 5 minutes, incluant le vidage du cache.

3
Collez le snippet avant </body>
Utilisez Ctrl+F pour trouver </body>. Collez le snippet SEEKR juste avant. Cliquez sur Sauvegarder.
<!-- SEEKR GPS Sémantique -->
<div id="seekr-search"></div>
<script async
  src="https://api.seekr-search.fr/widget/seekr-widget.js"
  data-key="sk_seekr_live_VOTRE_CLE_API"
  data-backend="https://api.seekr-search.fr"
  data-theme="dark-gold"

💡 Avantage SEEKR sur Shopify : La recherche Shopify native est limitée aux titres de produits et collections. SEEKR comprend les descriptions, les tags, les pages CMS et les articles de blog — et pointe directement vers le bon paragraphe.
🔌 Compatible avec : Tous les thèmes Shopify (Dawn, Debut, Brooklyn, Narrative, Pipeline, etc.), Shopify Plus.

Intégration Wix

Compatible avec Wix Editor et Wix Studio — via l'outil "Embed HTML".

Ouvrez votre éditeur Wix

2
Ajoutez un élément "Code HTML"
Dans l'éditeur, cliquez sur + Ajouter des éléments → Embed → HTML Code. Placez le widget dans une zone visible (par exemple dans le footer de votre site).
3
Collez le snippet SEEKR
Double-cliquez sur l'élément HTML Code que vous venez d'ajouter. Dans la fenêtre qui s'ouvre, collez le snippet SEEKR et cliquez sur Mettre à jour.
<!-- SEEKR GPS Sémantique -->
<div id="seekr-search"></div>
<script async
  src="https://api.seekr-search.fr/widget/seekr-widget.js"
  data-key="sk_seekr_live_VOTRE_CLE_API"
  data-backend="https://api.seekr-search.fr"
  data-theme="dark-gold"
  data-gps="true"

Publiez et lancez le crawl

⚠️ Note Wix : L'option "Code HTML" n'est disponible que sur les plans Wix payants (Core, Business ou supérieur). Le plan gratuit Wix ne permet pas d'intégrer du code tiers.

HTML statique
/ Next.js / Nuxt

Pour les sites HTML statiques, Next.js, Nuxt, Gatsby, Astro et tout framework JavaScript.
1
HTML statique — dans votre fichier HTML
Ouvrez le fichier HTML de votre page (ou votre template layout si vous utilisez un générateur statique) et collez le snippet juste avant </body>.
<!-- SEEKR GPS Sémantique -->
<div id="seekr-search"></div>
<script async
  src="https://api.seekr-search.fr/widget/seekr-widget.js"
  data-key="sk_seekr_live_VOTRE_CLE_API"
  data-backend="https://api.seekr-search.fr"
  data-theme="dark-gold"
  data-gps="true"
></script>
2
Next.js — dans _document.js ou layout.js
Pour Next.js, ajoutez le snippet dans votre composant _document.js (Pages Router) ou layout.js (App Router) :
// app/layout.js (App Router)
export default function RootLayout({ children }) {
  return (
    <html><body>
      {children}
      <div id="seekr-search"></div>
      <Script src="https://api.seekr-search.fr/widget/seekr-widget.js"
        data-key="sk_seekr_live_VOTRE_CLE"
        data-gps="true" strategy="lazyOnload" />
    </body></html>
  )
}
3
Lancez le crawl depuis votre dashboard
Une fois le widget visible sur votre site, retournez dans votre dashboard SEEKR et cliquez sur Lancer le crawl avec l'URL de votre site.
💡 Paramètres du widget disponibles :

data-theme : "dark-gold" | "light"
data-placeholder : texte du champ
data-max-results : nombre de résultats (défaut: 8)
data-gps : "true" active l'ancrage profond
data-backend : URL de l'API SEEKR
🔌 Frameworks compatibles : HTML5, Next.js 13/14, Nuxt 3, Gatsby, Astro, SvelteKit, Remix, Vue.js, React.
Questions d'intégration

Problèmes courants & solutions

Le widget ne s'affiche pas après l'intégration, que faire ? +
Vérifiez d'abord que votre clé API est correcte (elle commence par "sk_seekr_live_"). Ouvrez la console de votre navigateur (F12) — s'il y a une erreur 403, votre domaine n'est pas encore enregistré dans votre dashboard. Ajoutez votre domaine dans Paramètres → Sites → Nouveau site. Si l'erreur persiste, contactez le support.
Le crawl s'est terminé mais les résultats de recherche sont vides, pourquoi ? +
Le crawl peut prendre quelques minutes selon la taille de votre site. Attendez 5 minutes après la fin du crawl avant de tester. Si les résultats restent vides, vérifiez que votre site est accessible publiquement (pas en maintenance) et que le sitemap.xml est accessible à l'URL /sitemap.xml. Relancez un crawl depuis le dashboard.
Puis-je changer les couleurs du widget pour qu'il corresponde à mon site ? +
Oui. Le paramètre data-theme accepte "dark-gold" (noir et or, thème SEEKR par défaut) et "light" (fond blanc). La personnalisation avancée des couleurs (couleur principale, police, bordures) est disponible sur le plan Agency via votre dashboard. Contactez le support pour connaître les options CSS override.
Est-ce que SEEKR ralentit mon site ? +
Non. Le script SEEKR est chargé avec l'attribut "async" — il ne bloque pas le rendu de votre page. Le widget s'initialise en arrière-plan et n'impacte pas votre score Core Web Vitals. Le temps de réponse de l'API est inférieur à 200ms pour les requêtes de recherche.
Comment mettre à jour le contenu indexé après une modification de mon site ? +
Depuis votre dashboard SEEKR, allez dans la section de votre site et cliquez sur "Relancer le crawl". SEEKR re-crawle votre site et met à jour l'index. Pour les mises à jour fréquentes, vous pouvez aussi utiliser l'API d'indexation manuelle pour pousser du contenu en temps réel sans déclencher un crawl complet.
